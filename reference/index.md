# Package index

## Model fitting

- [`htlr()`](https://longhaisk.github.io/HTLR/reference/htlr.md) : Fit a
  HTLR Model
- [`htlr_prior()`](https://longhaisk.github.io/HTLR/reference/htlr_prior.md)
  : Generate Prior Configuration

## Prediction and inference

- [`predict(`*`<htlr.fit>`*`)`](https://longhaisk.github.io/HTLR/reference/predict.htlr.fit.md)
  : Make Prediction on New Data
- [`as.matrix(`*`<htlr.fit>`*`)`](https://longhaisk.github.io/HTLR/reference/as.matrix.htlr.fit.md)
  : Create a Matrix of Markov Chain Samples
- [`nzero_idx()`](https://longhaisk.github.io/HTLR/reference/nzero_idx.md)
  : Get Indices of Non-Zero Coefficients
- [`summary(`*`<htlr.fit>`*`)`](https://longhaisk.github.io/HTLR/reference/summary.htlr.fit.md)
  : Posterior Summaries
- [`evaluate_pred()`](https://longhaisk.github.io/HTLR/reference/evaluate_pred.md)
  : Evaluate Prediction Results

## Generating simulation data

- [`gendata_FAM()`](https://longhaisk.github.io/HTLR/reference/gendata_FAM.md)
  : Generate Simulated Data with Factor Analysis Model
- [`gendata_MLR()`](https://longhaisk.github.io/HTLR/reference/gendata_MLR.md)
  : Generate Simulated Data with Multinomial Logistic Regression Model

## Utility functions

- [`std()`](https://longhaisk.github.io/HTLR/reference/std.md) :
  Standardizes a Design Matrix
- [`split_data()`](https://longhaisk.github.io/HTLR/reference/split_data.md)
  : Split Data into Train and Test Partitions
